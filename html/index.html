get /history.json
Return daily historical coin data (historyJsonGet)
Returns daily historical coin data. History is written at the first `5 minutes` of each day. Server timezone is `UTC+01:00`. If the `period` param is set to `14days` it returns hourly data from the last 14 days.

Query parameters

coin (required)
Query Parameter — Coin - bitcoin
period (optional)
Query Parameter — Defaults to current year. `14days` is also an option default: 2017
format (optional)
Query Parameter — Can be `array` or `hash` default: array

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Returns the history

400

Format not supported Invalid coin Invalid year No history in year

get /ticker.json
Return coin ticker (tickerJsonGet)
Returns the timestamp, info about the coins, currency exchange rates and global data.

Query parameters

symbol (optional)
Query Parameter — Market or array of markets `BTC,DOGE`
identifier (optional)
Query Parameter — Market or array of markets `bitcoin,dogecoin`
page (optional)
Query Parameter — Page number default: 0.0
size (optional)
Query Parameter — Number of items per page default: 20.0

Produces

This API call produces the following media types according to the Accept request header; the media type will be conveyed by the Content-Type response header.
  • application/json

Responses

200

Returns the coin ticker
